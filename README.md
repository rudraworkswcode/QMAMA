# Q-MAMA
Routine data collection at Primary Health Centers (PHCs) for pregnant mothers often produces low-dimensional datasets. They contain maternal demographics, prenatal factors, clinical observations, gestational age, and blood reports, which are sparse data points and cannot be used effectively to serve as markers for early prediction of maternal complications. This project proposes a Hybrid Quantum-Classical Machine Learning Framework. We utilize Quantum Support Vector Machines (QSVM) and Quantum Kernel Methodologies, allowing for accurate predictions of maternal health risks using these datasets. Monthly data collections are first converted to regularized datasets. Classical Preprocessing Software is used for Classical Dimensionality Reduction. The preprocessed dataset is supplied to the Quantum Machine. It maps vectors into a high-dimensional quantum space. Quantum Kernels are used to map patient profiles. A Quantum Support Vector Machine is used to determine an optimal hyperplane, allowing for accurate predictions to occur. Using all of these techniques, it is possible for this system to accurately provide disease predictions for pregnant mothers. By adopting hybrid Quantum-Classical technologies, we can propose a system that integrates into existing healthcare frameworks, providing high-value diagnostic insights derived entirely from monthly clinical appointments at PHCs.
